给你一个二叉搜索树的根节点 root ，返回 树中任意两不同节点值之间的最小差值 。

示例 1：

输入：root = [4,2,6,1,3]

输出：1

示例 2：

输入：root = [1,0,48,null,null,12,49]

输出：1

提示：



树中节点数目在范围 [2, 100] 内

0 <= Node.val <= 10^5



题目分析：

1，根据二叉搜索树的性质，我们可以采取中序遍历的方式获取排序后的结果

2，由于节点的值在[0,10^5]范围内，节点值的差值在[-10^5-1,10^5+1]范围内

3，需要用一个pre记录前驱节点

4，针对二叉搜索树类型的题目，通过遍历可以得到有序的数列，然后可以求差值

5，根据题意，默认是升序
